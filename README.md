# Burger's Factory
Burger recipies editor react app

## Setup and run
```bash
# Download repository:
git clone https://github.com/sendgn/burger-factory.git
# Go to the app:
cd burger-factory
# Install dependencies:
npm install
# Production build - output will be at build/ folder
npm build
# Start dev server at http://localhost:3000/
npm start
```
